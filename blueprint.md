# Blueprint: Vita King Kong Ultra Energy Drink Landing Page

## Overview

This project is a visually striking, responsive landing page for a fictional product, "Vita King Kong Ultra Energy Drink." The goal is to create an immersive and energetic promotional experience using modern web technologies.

## Design & Features

### Initial Version (v1.0)

*   **Core Structure:**
    *   `index.html`: Main HTML document.
    *   `style.css`: Stylesheet for all visual presentation.
    *   `main.js`: Placeholder for future JavaScript interactivity.

*   **Visual Design & Aesthetics:**
    *   **Theme:** Primal energy, power, and intensity, inspired by the "King Kong" name.
    *   **Background:** A full-screen, auto-playing, muted video of a powerful gorilla serves as the primary visual hook.
    *   **Color Palette:** A high-contrast palette of electric green, dark charcoal, and white to create a vibrant, energetic feel.
    *   **Typography:**
        *   Uses the "Poppins" font from Google Fonts for a clean, modern look.
        *   Employs a strong typographic hierarchy with large, bold headlines and clear, legible body text.
        *   Text has a multi-layered drop shadow to create a sense of depth and lift it off the video background.
    *   **Layout:**
        *   A responsive, mobile-first design using Flexbox for centering and layout management.
        *   A translucent overlay on the video ensures text legibility.
        *   A subtle noise texture is applied to the background overlay for a premium, tactile feel.
    *   **Iconography:** Includes a custom-designed SVG logo for the "Vita King Kong" brand.

*   **Key Components & Interactivity:**
    *   **Header:** A simple header containing the brand logo.
    *   **Hero Section:** The main content area featuring the product name, a compelling tagline ("Ignite Your Primal Power"), and a call-to-action.
    *   **Call-to-Action (CTA) Button:** A brightly colored button with a "glow" effect on hover to draw user attention.

*   **Accessibility (A11Y):**
    *   Semantic HTML (`<header>`, `<main>`, `<h1>`, etc.) is used to structure the page correctly.
    *   The `playsinline` attribute is added to the video for better mobile browser compatibility.
    *   High-contrast text and a clear visual hierarchy aid readability.

## Current Plan

*   **Objective:** Transform the initial "Hello, world!" template into the Vita King Kong promotional landing page.
*   **Steps:**
    1.  **HTML (`index.html`):**
        *   Replace the entire body content.
        *   Add a `<header>` with an SVG logo.
        *   Create a `<main>` section for the hero content.
        *   Embed a full-screen background video.
        *   Add headline, tagline, and CTA button elements.
        *   Update the `<title>` tag.
        *   Link to Google Fonts.
    2.  **CSS (`style.css`):**
        *   Implement a complete styling overall.
        *   Style the video background to be full-screen and covered.
        *   Add the background overlay and noise texture.
        *   Style the typography, including font sizes, colors, and drop shadows.
        *   Style the CTA button with hover effects.
        *   Add media queries to ensure responsiveness on mobile devices.
    3.  **JS (`main.js`):**
        *   Clear the existing content. The page will be static initially, so no JS is required for v1.0.
